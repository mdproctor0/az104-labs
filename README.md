## The Scenario

All labs in this series are built around **Contoso Financial Group** — a fictional mid-sized financial services company migrating to Azure. Each lab solves a real business requirement across the five AZ-104 exam domains.

No tutorials. No walkthroughs. Requirements given, solution built.
